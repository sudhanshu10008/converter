# **how to use**

## for linux
first you need to install python in your computer  
go to --> https://www.python.org/  
download and install it  
<br>
- update 
  ```
  sudo apt update
  ```
- Open terminal and install git

    ```
    sudo apt install git
    ```
- Clone repo in your pc
  
    ```
    git clone 
    ```
- open directory
    ```
    cd converter
    ```
- run converter.py
  ```
  python converter.py
  ```
